# relay status

Check daemon health and connection state.

```bash
relay status
relay status --json
```

## Flags

| Flag | Description |
|------|-------------|
| `--json` | Output as machine-parseable JSON |

Returns: daemon PID, uptime, WhatsApp connection state, active instance count, total instance count.
